## Acceptance Criteria & Specs

### Button (AT-01)
- Props: `variant?: 'primary' | 'secondary' | 'ghost' | 'danger'`, `size?: 'sm' | 'md' | 'lg'`, `loading?: boolean`, `fullWidth?: boolean`, `disabled?: boolean`, `children?: React.ReactNode`. Extends `React.ButtonHTMLAttributes<HTMLButtonElement>`.
- `forwardRef` to `<button>`.
- Height: `sm`=32px, `md`=40px, `lg`=48px.
- `loading=true`: Internal CSS spinner + `aria-busy="true"` + automatically disabled. (Do NOT import Spinner atom - atoms are indivisible).
- Disabled: opacity 0.45, cursor not-allowed.
- Focus visible: box-shadow 0 0 0 3px var(--color-primary-100).
- `margin: 0`.

### Input (AT-02)
- Props: `label?: string`, `error?: string`, `hint?: string`, `prefix?: React.ReactNode`, `suffix?: React.ReactNode`. Extends `React.InputHTMLAttributes<HTMLInputElement>`.
- `forwardRef` to `<input>`.
- Autogenerated `id` via `useId()` for `htmlFor`/`id` linking.
- Wrapper `div` with border that handles `:focus-within` (not input directly).
- Error state: red border (`var(--color-danger-600)`), `aria-invalid="true"`, `aria-describedby` pointing to error text paragraph with `role="alert"`.
- `hint` hidden when error is present.
- `margin: 0`.
